About OTN APP

Travel industry is developing at an unprecedented rate, and due to the constant emergence of new applications. This Progressive Web Apps which can be used when there is no internet, which brings great convenience to tourist. The motivation for this project is that most travel apps on the market cannot be used without the Internet, and most tourist do not have the habit of searching for travel guide information, so they often get lost when traveling in an unfamiliar environment. This is cumbersome for tourist and spoils the mood.

Therefore, "OTN" (Offline Travel Navigation) application is a solution for tourist, it provide a integrated functions of all other featured applications in one application. The proposed project use the currently popular open source technology for development. With the help of these technique, the proposed application is expected to achieve offline use and provide excellent user experience and high-speed performance.
